Description
-------------------------
Spiral DBAL for Symfony

Feature:
-------------------------
- Injection of a specific database 
- Support debugger (Logging query)

Requirements
-------------------------
- php >= 7.2
- symfony >= 4.0

## Installation

The package could be installed with composer:

```
composer install myaza-software/spiral-database-bundle
```
